## 설치 방법
### node, npm 설치

Homebrew를 설치 후 아래 명령어를 통해 설치합니다.
```sh
brew update
brew install node
node -v
npm -v
```

### yarn 설치
yarn 또한 homebrew를 이용하여 아래의 명령어로 설치합니다

```sh
brew install yarn --ignore-dependencies
yarn -v
```
